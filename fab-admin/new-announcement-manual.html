Please read the wiki entry first before updating or creating announcements as it can result in turning off delivery dates.

There are two different announcement types, Manual and Auto.

Manual messages will be displayed at the set times and can also block delivery dates (if set). Auto announcements are triggered by order limits - once an order limit is reached for the set day, the delivery date will be blocked and a message will be displayed (messages are displayed up to 7 days in advance).
